# Installation and Execution

In command terminal, enter:
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

To run, enter:
```
uvicorn main:app --reload
```

In your web browser, open http://127.0.0.1:8000 and login with:
```
user1@gmail.com
12345
```
